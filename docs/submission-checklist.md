# Submission Checklist

## 基础材料

- [ ] 报名表 1 份
- [x] 项目研究报告
- [ ] 项目查新报告
- [ ] 查新附件清单
- [ ] PPT
- [ ] 作品视频
- [ ] 作品主页或运行入口

## 签字盖章

- [ ] 申报者签名
- [ ] 指导教师签名
- [ ] 学院盖章
- [ ] 查新声明

## 证据材料

- [ ] 实验指标截图或表格
- [ ] 部署或测试记录
- [ ] 用户反馈或应用证明
- [ ] 论文、软著、专利、开源地址
- [ ] 原始数据或样例数据说明

## 最终检查

- [ ] 项目简介 400 字以内
- [ ] 创新点与查新点一一对应
- [ ] 研究报告、查新报告页数已填写
- [ ] 文件命名符合最终通知
- [ ] `submission/` 中只保留最终提交材料
