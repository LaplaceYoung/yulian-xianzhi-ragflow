# 御链先知前端入口

## 页面目标

制作面向评委的作品主页和智能合约攻击检测产品界面，突出交易哈希分析、函数调用链、RAGFlow 语义推理和风险报告能力。

## 内容来源

- `submission/【项目报告】御链先知.pdf`
- `docs/project-brief.md`
- `docs/competition-form-outline.md`
- `docs/novelty-search-outline.md`
- `docs/submission-checklist.md`

## 页面模块

1. 首页首屏：作品名、智能合约安全痛点、跨模态语义增强价值、核心指标。
2. 风险分析控制台：交易哈希输入、链上信息解析、检测进度、风险输出。
3. 调用链图谱：合约地址、函数节点、跨合约调用、状态变化。
4. 语义检索：Top-k 相似攻击路径、静态语义资料、Prompt 上下文。
5. 风险报告：漏洞类型、风险等级、攻击路径、处置建议。
6. 指标对比：Slither、TXSpector、SmartKG、GPT-4、RAGFlow 性能对比。

## 视觉关键词

区块链安全、交易追踪、调用图谱、语义检索、风险研判、智能审计。
